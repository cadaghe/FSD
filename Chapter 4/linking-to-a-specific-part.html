Structure

Think about the stories you read in a newspaper: for each story, there will be a headline, some text, and possibly someimages. If the article is a long piece, there may be subheadings newspaper.

white space collapsing

When the browser comes across two or more spaces next to each other, it only displays one space. Similarly if it comes across a line break, it treats that as a single space too. This is known as white space collapsing.

Headings

Headings are created by highlighting text then using a drop-down box to select a heading.
